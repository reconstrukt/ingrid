Modern Full Options

By default, Modern Grid returns a jQuery object - by assigning a variable to the return, you can access the following: 

  HTML BLOCKS: 

    infoBarTLeft
    infoBarTCenter
    infoBarTRight
    infoBarBLeft
    infoBarBCenter
    infoBarBRight
  
  FUNCTIONS:

    clearSelected
    getSelectedRows
    updatePaging
    closeRightClickMenu
    refresh
    reload

Below find the list of options available for Moder Grid. 

    /*  Basic Table Config */
    tableType : 'static',                           // type of table - static, hybrid, or dynamic

    colWidths: [],                                  // array of column widths eg [120, 120, 100, 100]
    minColWidth: 20,                                // minimum column width Column width will not go below this number
    target: null,                                   // the div that will hold the finished product
