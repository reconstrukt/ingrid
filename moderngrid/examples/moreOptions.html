Modern Grid Ajax Example - 3

Below are a few different examples of interactive data. I've added in Bootstrap to show how Modern Grid works with a modern front end framework.

We are now sorting, and adding in a search as well. Both tables are now shown together (you can have multiple modernGrids per page). Note the additional options. 

<div class="mainTable">
</div>

<script type="text/javascript">
$(function() {
  var options = {
    url: "ajaxExamples/sortInlineClasses.php",
    tableType: "dynamic",
    sorting: true,
    sortDefaultDir: 'asc',
    recordsPerPage: 2,
    pageNumber: 1, 
    infoBarBottom: true,
    infoBarTop: true,
    paging: { show: true, position: "bl"}, 
    search: { show: true, position:"tr"},
    pageToolbarHeight: 35,
    defaultSortColClass: "id",
    rowCount: { show: true, position: 'br'},
    totalRecords: 6,
  };

  $.modernGrid(".mainTable", options);
</script>
