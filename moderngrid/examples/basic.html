Modern Grid Basic Example

Modern Grid is a jQuery data grid plugin, used to turn static tables into fully interactive tools. This page shows the bare minimum required to use Modern Grid - the only styling used is to help highlight code sections in order to show the difference between plain HTML and Modern Grid.

The first step is to include jQuery (jQuery) - you can host jQuery locally, or by CDN (jQuery CDN). To keep things simple, I'll host it locally in all examples: 

<script type="text/javascript" src="jQuery/jquery-3.0.0.js"></script>
Once jQuery has been included, include modernGrid.js and modernGrid.css: 
<script type="text/javascript" src="js/modernGrid.js"></script>
<link rel="stylesheet" type="text/css" href="css/modernGrid.css" />

You are now ready to use Modern Grid on your tables!

Let's start with a basic table - in this case, a meal plan covering Monday - Friday, Breakfast, Lunch & Dinner.

Monday Tuesday Wednesday Thursday Friday
Pancakes Waffles Bacon & Eggs Toast Cereal
Ham & Cheese Sandwich Salami Sandwich Roast Beef Sandwich Turkey Sandwich PB & J Sandwich
Hambugers Pizza Pasta Tacos Steak

To use modernGrid, put the table in a div, and give the div a class. This div now becomes our target div - we pass it into modernGrid and let it do the work: 

<div class="mainTable">
...table code
</div>

<script type="text/javascript">
  $(function() {
    $.modernGrid(".mainTable");
 });
</script>
And the result:

Monday Tuesday Wednesday Thursday Friday
Pancakes Waffles Bacon & Eggs Toast Cereal
Ham & Cheese Sandwich Salami Sandwich Roast Beef Sandwich Turkey Sandwich PB & J Sandwich
Hambugers Pizza Pasta Tacos Steak

While Modern Grid can be used to quickly style inline tables, the real power comes from its ability to update data through ajax.
