Modern Grid Ajax Example - 1

In this example, we'll use the same table as in the basic example, but retrieve it through ajax. The ajax page is the basic table, with no additional div required.

This time, we start with an empty div in the body of our page. In order to tell Modern Grid that it will be retriving the data itself, we will have to specify a few options: 

<div class="mainTable">
</div>

<script type="text/javascript">
$(function() {
  var options = {
    url: "ajaxExamples/basic.html",
    tableType: "dynamic",
  };

  $.modernGrid(".mainTable", options);
</script>
